Here's a miscellany of other Gosu features you might find useful:

Object Initializers

You can set properties of an object in its constructor by using the object initializer syntax: 

  var newObj = new MyClass() {
    :Prop1 = "Foo",
    :Prop2 = "Bar"
  }
This will construct the MyClass object and then set the Prop1 and Prop2 properties.
