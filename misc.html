Misc

Here's a miscellany of other Gosu features you might find useful:

Object Initializers

You can set properties of an object in its constructor by using the object initializer syntax: 

  var newObj = new MyClass() {
    :Prop1 = "Foo",
    :Prop2 = "Bar"
  }
This will construct the MyClass object and then set the Prop1 and Prop2 properties.

Named Arguments

Gosu supports named arguments if two conditions are met:

  • When the method (or constructor) comes from a source that provides good naming information. (Java classes, for example, do not.)
  • The method (or constructor) is not overloaded
If these two conditions are met, you can use named arguments. Let's look at an example: 
  function printIt( msg:String, escapeIt:boolean, printDate:boolean) {
    if( escapeIt ) {
      msg = msg.replace( "<", "&lt;" )
    }
    if( printDate ) {
      msg = (new Date()) + " " + msg
    }
    print( msg )
  }
  
  printIt( "<b>Hello World</b>", true, true ) // what does that mean?

  printIt( :msg = "<b>Hello World</b>", 
           :escapeIt = true, 
           :printDate = true )                // Oh.
When methods take lots of arguments, or lots of non-descript arguments like booleans, named arguments can help clarify your code dramatically.
